# hapi-boilerplate
A very simple structured HAPI boilerplate with mongodb, cors and token base authentication.

```
setup
cd hapi-boilerplate
npm install
npm start
```

```
Sample routes
http://localhost:8080
```

```
Sample routes with auth
http://localhost:8080/api/test
http://localhost:8080/api/test?access_token=1234
```
